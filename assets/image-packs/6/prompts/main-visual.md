# Main Visual

Generate the full finished magazine-style image described by the source prompt. Keep the character in the scene.

Common constraints:
- no real brand logos
- no watermark
- no unreadable large text blocks
- no random numbers, page numbers, issue numbers, prices, years, dates, rankings, serial numbers, barcode numbers, or meaningless numeric text
- keep the same character identity, outfit logic, color palette, and art direction from the source prompt


## Source prompt

## 1. 图像类型与角色锁定

请创作一张高端时装专题杂志内页风格的动漫插画，主角是《我的青春恋爱物语果然有问题》中的雪之下雪乃。

这张图要像真实高级时装杂志中的 atelier 专题页面，而不是普通角色展示图。画面需要有服装设计稿、编辑版式、工作室空间和印刷质感。

## 2. 人物特征与气质表情

她有着黑色长直发、白皙肤色、精致清秀的五官、蓝灰色眼睛，整体气质要体现出理性、专业、优雅、克制、清冷而从容。

表情可以是平静专注、略带淡淡微笑，像正在参与一次服装试装或设计访谈。人物必须保持雪之下雪乃的辨识度，不要改变发色、眼色、脸型、发型长度或整体角色身份。

## 3. 杂志风格参考与主题方向

整体风格参考 SO-EN、装苑式服装设计专题和高级时装编辑杂志的视觉气质，强调版型、面料、结构、剪裁、服装工作室和设计过程。

本页主题是“白色 atelier 与服装结构”。画面可以包含布料样本、纸样、衣架、裁缝台、半透明窗帘、白墙、线稿和少量服装说明框，氛围专业、干净、有设计感。

## 4. 服装造型与鞋履禁忌

服装采用结构感强的现代都市造型：短款象牙白西装、雾蓝高领细针织、炭黑高腰阔腿裤或长款直筒裙、细腰带、银色耳饰、简洁手表或细链项链。

整体搭配要突出服装的剪裁线、肩线、腰线和面料厚薄变化。鞋子必须是平底乐福鞋、平底牛津鞋、低矮方跟短靴或简洁平底皮鞋。禁止高跟鞋、细跟鞋、尖头高跟鞋、恨天高、夸张厚底鞋和过度性感鞋履。

## 5. 色彩、材质与季节感

色彩搭配建议以象牙白、冷灰、炭黑、雾蓝、淡银色为主，局部可以加入浅米色布料样本或很少的淡金色金属细节。

材质需要清楚：挺括西装面料、细密针织、哑光皮革、棉麻样布、半透明窗帘、白色纸样和磨砂金属。整体不要只有白灰，需要用雾蓝和淡金属光让画面更有层次。

## 6. 构图、背景与空间

画面采用竖版 9:16 或宽高比约 0.56 的杂志页面构图，适合 1200x2135 或更高分辨率。人物可以全身站在裁缝台旁，也可以七分身坐在工作室椅子上。

背景是白色服装工作室或摄影棚，空间简洁但有细节。画面需要保留大块留白用于标题、面料说明和小图注，不要让工具和衣架过度杂乱。

## 7. 光线、姿态与镜头质感

光线采用高级摄影棚 editorial 柔光，面部阴影自然，衣服边缘和布料纹理清楚，整体有服装画册般的干净层次。

人物姿态要端正、专业、优雅，可以轻扶袖口、整理衣领、站在半成衣旁或看向镜头。不要夸张摆拍，不要过度性感，不要把她画成舞台模特。

## 8. 杂志版式、文字限制与最终效果

版式上加入杂志内页排版元素：服装专题标题、面料说明框、细线分隔、栏目标签、英文短句和日文短句。可以使用虚构文字，例如“FORM AND RESTRAINT”“ATELIER NOTE”“TAILORED SILHOUETTE”“静かな構造美”。

注意这是虚构时尚杂志，不要使用真实杂志 logo，不要真实品牌标志，不要水印，不要乱码，不要难以辨认的大段文字。整体效果要突出：雪之下雪乃、服装结构、白色 atelier、高级时装专题、清冷理性、平底鞋、无高跟鞋、真实杂志内页质感。

返工硬约束：不要出现任何随机数字、页码、期号、价格、年份、月份、日期、排名、序号、列表编号、条形码数字或无意义数字。侧边信息栏需要秩序感时，用短横线、色块、图标、无编号小标题和非数字项目符号替代。不要出现 01、02、03、Vol、No、2025、2026、980 yen 等数字文本。

镜头角度与姿势变化：人物坐在工作室椅子或裁缝台旁，身体侧向画面，脸转向镜头，一手整理袖口或轻触布料样本。
