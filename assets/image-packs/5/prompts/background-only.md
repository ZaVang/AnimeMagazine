# Background Only

Generate the same page, scene, or magazine layout without any person.

Reconstruct any area that would normally be hidden behind the character. Preserve layout, shadows, page texture, background architecture, material panels, color blocks, and editorial composition. Hard constraint: no human body parts, no silhouette, no ghosting, no hair, no hand, no shoes, no bag, no clothing fragments.

Common constraints:
- no real brand logos
- no watermark
- no unreadable large text blocks
- no random numbers, page numbers, issue numbers, prices, years, dates, rankings, serial numbers, barcode numbers, or meaningless numeric text
- keep the same character identity, outfit logic, color palette, and art direction from the source prompt


## Source prompt

## 1. 图像类型与角色锁定

请创作一张高质感日系文化时尚杂志内页风格的动漫插画，主角是《我的青春恋爱物语果然有问题》中的雪之下雪乃。

这张图要像真实出版物中的一页，而不是普通角色海报或手机壁纸。画面需要有完整的时尚编辑感、生活方式杂志气质和高级纸张印刷质感，适合作为 3D 翻页杂志的内页素材。

## 2. 人物特征与气质表情

她有着黑色长直发、白皙肤色、精致清秀的五官、蓝灰色眼睛，整体气质要体现出知性、优雅、理性、克制、清冷而高级。

表情自然平静，可以带一点淡淡微笑，像在安静地观察书架或刚从阅读中抬头。人物必须保持雪之下雪乃的辨识度，不要改变发色、眼色、脸型、发型长度或整体角色身份，不要画成其他黑发少女。

## 3. 杂志风格参考与主题方向

整体风格参考 FUDGE、GINZA 和日系城市文化杂志的视觉气质，强调东京书店、轻学院风、日常时装、文艺感和安静高级。

本页主题是“东京书店与春日阅读”。画面可以围绕独立书店、杂志架、玻璃窗、木质书桌、咖啡杯、植物影子和柔和街景展开。整体要像一篇关于书、服装与生活习惯的时尚专题内页。

## 4. 服装造型与鞋履禁忌

服装采用现代日系学院派都市穿搭：白衬衫、细针织背心或短款开衫、深海军蓝百褶长裙或高腰直筒裤、浅灰风衣、窄皮带、帆布托特包、银色小耳饰。

整体搭配要有轻松但讲究的层次，突出面料柔软度、衣服轮廓和书卷气。鞋子必须是平底乐福鞋、平底牛津鞋、平底玛丽珍鞋或干净平底短靴。禁止高跟鞋、细跟鞋、尖头高跟鞋、恨天高、夸张厚底鞋和过度性感鞋履。

## 5. 色彩、材质与季节感

色彩搭配建议以象牙白、深海军蓝、浅灰、木色、雾蓝和少量苔绿色为主，整体低饱和、清爽、温和，有春日下午的安静空气感。

材质需要清楚：棉质衬衫、细腻针织、挺括风衣布、哑光皮革、纸张纹理、木质书架、玻璃反光和杂志纸颗粒。画面不要一味蓝灰，可以让木色和苔绿色带出生活感。

## 6. 构图、背景与空间

画面采用竖版 9:16 或宽高比约 0.56 的杂志页面构图，适合 1200x2135 或更高分辨率。人物可以是全身或七分身，站在书架与窗边之间，手里拿着一本杂志或轻扶书脊。

背景是安静的东京独立书店或杂志店，有清晰但不喧闹的书架、封面陈列、窗光和留白。画面需要预留标题、图注和小栏目空间，不要让背景细节挤满整页。

## 7. 光线、姿态与镜头质感

光线采用春日下午的柔和自然光，从窗边斜斜照入，形成干净的阴影和细腻的纸面质感。

人物姿态要安静、端正、自然，像真正的日系时尚杂志书店专题模特。她可以低头看书、轻轻回望镜头或站在杂志架前，但不要夸张摆拍，不要过度动态，不要过度甜美。

## 8. 杂志版式、文字限制与最终效果

版式上加入杂志内页排版元素：专题标题、细线分隔、图注块、栏目名、少量英文短句和日文短句。可以使用虚构文字，例如“BOOKS AND QUIET MODE”“YUKINO YUKINOSHITA”“SPRING READING”“静けさを読む”。

注意这是虚构时尚杂志，不要使用真实杂志 logo，不要真实品牌标志，不要水印，不要乱码，不要难以辨认的大段文字。整体效果要突出：雪之下雪乃、东京书店、日系文化杂志、文艺学院感、春日清爽、平底鞋、无高跟鞋、真实杂志内页质感。

返工硬约束：不要出现任何随机数字、页码、期号、价格、年份、月份、日期、排名、序号、列表编号、条形码数字或无意义数字。侧边信息栏需要秩序感时，用短横线、色块、图标、无编号小标题和非数字项目符号替代。不要出现 01、02、03、Vol、No、2025、2026、980 yen 等数字文本。

镜头角度与姿势变化：人物采用书店内轻微俯拍角度，身体三分之二侧向书架，脸从书页或杂志上抬起看向镜头。
