# Background Only

Generate the same page, scene, or magazine layout without any person.

Reconstruct any area that would normally be hidden behind the character. Preserve layout, shadows, page texture, background architecture, material panels, color blocks, and editorial composition. Hard constraint: no human body parts, no silhouette, no ghosting, no hair, no hand, no shoes, no bag, no clothing fragments.

Common constraints:
- no real brand logos
- no watermark
- no unreadable large text blocks
- no random numbers, page numbers, issue numbers, prices, years, dates, rankings, serial numbers, barcode numbers, or meaningless numeric text
- keep the same character identity, outfit logic, color palette, and art direction from the source prompt


## Source prompt

## 1. 图像类型与角色锁定

请创作一张巴黎与东京混合气质的高级时尚杂志内页动漫插画，主角是《我的青春恋爱物语果然有问题》中的雪之下雪乃。

这张图要像真实女性时尚杂志中的城市咖啡馆专题页面，而不是普通咖啡厅角色图。画面需要有成熟编辑感、服装造型、街景氛围和高级印刷质感。

## 2. 人物特征与气质表情

她有着黑色长直发、白皙肤色、精致清秀的五官、蓝灰色眼睛，整体气质要体现出知性、优雅、从容、克制、清冷但不过分疏离。

表情可以自然平静、带淡淡微笑，像在安静咖啡馆露台上等一场访谈。人物必须保持雪之下雪乃的辨识度，不要改变发色、眼色、脸型、发型长度或整体角色身份。

## 3. 杂志风格参考与主题方向

整体风格参考 Madame Figaro Japon、L'Officiel 和法式都市时尚 editorial 的视觉气质，强调咖啡馆、优雅外套、都市女性、轻奢但不张扬的成熟风格。

本页主题是“东京咖啡馆里的法式克制”。背景可以是街角咖啡馆、玻璃门、藤编椅、小圆桌、街树影子、细窄街道和温柔午后光。

## 4. 服装造型与鞋履禁忌

服装采用法式都市与日系克制结合的造型：短款粗花呢外套或轻薄呢料夹克、白衬衫、深色直筒长裤或及踝长裙、丝巾、细腰带、小号皮包、珍珠耳饰。

整体搭配要优雅、成熟、轻盈，不要复古过头，不要甜腻。鞋子必须是平底乐福鞋、平底玛丽珍鞋、平底芭蕾鞋或低矮方跟鞋。禁止高跟鞋、细跟鞋、尖头高跟鞋、恨天高、夸张厚底鞋和过度性感鞋履。

## 5. 色彩、材质与季节感

色彩搭配建议以奶油白、黑色、浅驼色、酒红、珍珠灰和少量雾蓝为主。整体有法式温度，但要保留雪乃的冷静感。

材质需要清楚：粗花呢、丝巾、棉质衬衫、哑光皮革、藤编椅、玻璃反光、咖啡瓷杯和纸张颗粒。酒红只作为小面积点缀，避免画面过暖。

## 6. 构图、背景与空间

画面采用竖版 9:16 或宽高比约 0.56 的杂志页面构图，适合 1200x2135 或更高分辨率。人物可以坐在小圆桌旁，也可以站在咖啡馆门口侧身回望。

背景是干净的城市咖啡馆或露台，桌椅、门框和街树形成自然版式。画面需要给标题、短引文、图注和栏目留白留出空间，不要让餐具和街景过度杂乱。

## 7. 光线、姿态与镜头质感

光线采用午后自然光，温柔但不甜腻，脸部光影自然，服装纹理、藤编椅和玻璃反光要清晰。

人物姿态要端庄、从容、安静，可以轻扶杯子、翻看小册子、整理丝巾、侧脸看向街景或画面外采访者。不要夸张摆拍，不要过度可爱，不要性感化。

## 8. 杂志版式、文字限制与最终效果

版式上加入女性时尚杂志内页元素：城市专题标题、短引文、细线分隔、图注块、英文与日文短句。可以使用虚构文字，例如“CAFE MODE”“PARIS IN TOKYO”“YUKINO YUKINOSHITA”“静けさのカフェ時間”。

注意这是虚构时尚杂志，不要使用真实杂志 logo，不要真实品牌标志，不要水印，不要乱码，不要难以辨认的大段文字。整体效果要突出：雪之下雪乃、法式都市、咖啡馆、酒红点缀、平底鞋、无高跟鞋、真实杂志内页质感。

返工硬约束：不要出现任何随机数字、页码、期号、价格、年份、月份、日期、排名、序号、列表编号、条形码数字或无意义数字。侧边信息栏需要秩序感时，用短横线、色块、图标、无编号小标题和非数字项目符号替代。不要出现 01、02、03、Vol、No、2025、2026、980 yen 等数字文本。

镜头角度、姿势与肢体自然度：人物坐在咖啡馆小圆桌旁，身体略向桌面倾斜，一只手轻扶无文字咖啡杯杯耳，另一只手整理丝巾或搭在椅背上。腿部可以自然交错放置，平底鞋完整可见。脸可以是柔和侧脸，视线看向街边玻璃反光或桌上的小册子，只让眼神从侧面露出一点，不必盯镜头。手指要贴合杯子和丝巾，肩颈放松，椅子、桌面和身体比例要真实，避免手臂过长、手掌变形或坐姿漂浮。
