# Main Visual

Generate the full finished magazine-style image described by the source prompt. Keep the character in the scene.

Common constraints:
- no real brand logos
- no watermark
- no unreadable large text blocks
- no random numbers, page numbers, issue numbers, prices, years, dates, rankings, serial numbers, barcode numbers, or meaningless numeric text
- keep the same character identity, outfit logic, color palette, and art direction from the source prompt


## Source prompt

## 1. 图像类型与角色锁定

请创作一张运动轻奢风格的日系时尚杂志内页动漫插画，主角是《我的青春恋爱物语果然有问题》中的雪之下雪乃。

这张图要像真实时尚杂志中的 clean sporty editorial 页面，而不是普通运动服角色图。画面需要有运动感、都市感、精致版式和高级印刷质感。

## 2. 人物特征与气质表情

她有着黑色长直发、白皙肤色、精致清秀的五官、蓝灰色眼睛，整体气质要体现出清爽、聪明、理性、克制、自信和轻盈。

表情自然放松，可以带一点淡淡微笑，像在清晨城市中完成一组轻运动穿搭拍摄。人物必须保持雪之下雪乃的辨识度，不要改变发色、眼色、脸型、发型长度或整体角色身份。

## 3. 杂志风格参考与主题方向

整体风格参考 ELLE Japon、Vogue Japan 的运动时装专题和现代 athleisure editorial 的视觉气质，强调轻运动、干净线条、功能面料、城市早晨和清爽女性力量。

本页主题是“清晨的运动轻奢”。背景可以是现代公园、玻璃健身房外侧、河岸步道、白色长椅、清晨建筑影子和开阔天空。

## 4. 服装造型与鞋履禁忌

服装采用运动与时装结合的造型：短款防风夹克、修身高领运动针织、宽松束脚长裤或直筒运动长裤、轻量马甲、简洁腰包或斜挎小包、银色耳饰。

整体搭配要干净、利落、有功能感，但不能变成普通健身服。鞋子必须是低帮平底运动鞋、干净德训鞋、平底帆布鞋或平底乐福鞋。禁止高跟鞋、细跟鞋、尖头高跟鞋、恨天高、夸张厚底鞋和过度性感鞋履。

## 5. 色彩、材质与季节感

色彩搭配建议以冷白、雾蓝、海军蓝、浅银灰、少量钴蓝或薄荷绿为主。整体明亮清爽，但不要过度荧光。

材质需要清楚：防风尼龙、弹力针织、哑光拉链、运动织带、金属扣、玻璃反光和纸张颗粒。钴蓝或薄荷绿只作为运动感点缀。

## 6. 构图、背景与空间

画面采用竖版 9:16 或宽高比约 0.56 的杂志页面构图，适合 1200x2135 或更高分辨率。人物可以全身站在河岸或公园边，也可以七分身靠近玻璃墙。

背景要开阔、干净、有空气感，利用栏杆、步道或建筑线条形成版式结构。画面需要预留运动专题标题、穿搭标签和图注空间。

## 7. 光线、姿态与镜头质感

光线采用清晨自然光，明亮、通透、略带冷调。人物面部要清晰，运动面料和发丝要有轻微风感。

人物姿态要轻盈、从容、自信，可以整理袖口、拿着水瓶、站在步道上或轻扶栏杆。不要跑步冲刺，不要夸张健身动作，不要性感化。

## 8. 杂志版式、文字限制与最终效果

版式上加入运动时装专题元素：小标签、功能面料说明、英文短句和日文短句。可以使用虚构文字，例如“CLEAN SPORT MODE”“MORNING ROUTINE”“YUKINO YUKINOSHITA”“軽やかな知性”。

注意这是虚构时尚杂志，不要使用真实杂志 logo，不要真实品牌标志，不要水印，不要乱码，不要难以辨认的大段文字。整体效果要突出：雪之下雪乃、运动轻奢、清晨城市、钴蓝薄荷点缀、平底运动鞋、无高跟鞋、真实杂志内页质感。

返工硬约束：不要出现任何随机数字、页码、期号、价格、年份、月份、日期、排名、序号、列表编号、条形码数字或无意义数字。侧边信息栏需要秩序感时，用短横线、色块、图标、无编号小标题和非数字项目符号替代。不要出现 01、02、03、Vol、No、2025、2026、980 yen 等数字文本。

镜头角度与姿势变化：人物站在清晨步道或栏杆旁，身体朝前方轻微行走，脸自然侧转看镜头，一手整理袖口或拿水瓶。
