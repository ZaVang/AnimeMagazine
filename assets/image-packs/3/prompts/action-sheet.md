# Action Difference Sheet

Generate a clean action/pose difference sheet for the same character, outfit, and art style.

Use a simple grid layout with separated full-body or upper-body pose variants. Prefer neutral stand, slight turn, hand to lapel, holding bag if the outfit includes one, gentle walk step, and looking aside. Keep the same character, same outfit, same lighting, and clear whitespace between variants. Labels are optional; if labels appear, use short words and no numbers.

Common constraints:
- no real brand logos
- no watermark
- no unreadable large text blocks
- no random numbers, page numbers, issue numbers, prices, years, dates, rankings, serial numbers, barcode numbers, or meaningless numeric text
- keep the same character identity, outfit logic, color palette, and art direction from the source prompt


## Source prompt

## 1. 图像类型与角色锁定

请创作一张高端奢华时尚杂志页面风格的动漫插画，主角是《我的青春恋爱物语果然有问题》中的雪之下雪乃。

这张图要像真实国际高级时装杂志的封面级专题页，而不是普通角色海报。画面需要有强烈主视觉、奢华质感、完整排版和侧边服装设计解析。

## 2. 人物特征与气质表情

她有着黑色长直发、白皙肤色、精致立体的五官、蓝灰色眼睛，整体气质要体现出高贵、冷艳、知性、优雅、克制、自信且具有强烈存在感。

表情不需要过于冷漠，可以是平静从容、略带淡淡微笑，但依然保留高级疏离感与强大气场。人物必须保持雪之下雪乃的辨识度，不要改变发色、眼色、脸型、发型长度或整体角色身份。

## 3. 杂志风格参考与主题方向

整体风格参考国际高级时装大刊、奢侈品广告大片和秋冬 couture editorial 的视觉气质。参考名只用于风格，不要出现在画面文字中。

本页主题是“冬季黑色长大衣与静默力量”。重点表现大衣轮廓、羊毛质地、皮革细节、珍珠或金属配饰，以及高级时装中的理性气场。

## 4. 服装造型与鞋履禁忌

服装选择极具高级感的秋冬造型：黑色或深海军蓝长款羊毛大衣、象牙白高领针织或丝质衬衫、深色长裙或高腰阔腿裤、皮质手套、珍珠耳饰、淡金或银色金属配饰。

整体穿搭要像高级时装秋冬系列造型，突出剪裁、轮廓、面料厚度和高级质感。鞋子必须是平底长靴、平底短靴、平底乐福鞋或低矮方跟靴。禁止高跟鞋、细跟鞋、尖头高跟鞋、恨天高、夸张厚底鞋和过度性感鞋履。

## 5. 色彩、材质与季节感

色彩搭配以象牙白、黑色、深海军蓝、冷灰、香槟金为主，局部可以加入少量酒红或深绿色作为奢华点缀。整体精致、利落、贵气，但不要一片黑。

材质需要清楚：羊毛大衣、细密高领针织、哑光皮革、珍珠、淡金金属、深色背景、柔和石材和纸张颗粒。侧边材料板可以展示羊毛、皮革、针织、金属、珍珠五种质感。

## 6. 构图、背景与空间

画面采用竖版 9:16 或宽高比约 0.56 的杂志页面构图，适合 1200x2135 或更高分辨率。人物可以全身站在极简摄影棚、石材空间或深色廊道中，让长大衣形成强轮廓。

背景使用高级摄影棚感背景或极简而富有层次的纯色背景。侧边或下方设置奢华时装解析栏，包含 “MATERIAL FOCUS”“WINTER COUTURE”“COORDINATE POINT”“THE NEW STANDARD” 等虚构栏目、面料小样、色彩板和穿搭说明。

## 7. 光线、姿态与镜头质感

光线采用高级时尚摄影棚布光，有明显但优雅的明暗层次，让脸部、黑发、大衣轮廓和皮革细节更加立体。

人物姿态要自信、从容、优雅，像真正的封面模特，具有强烈镜头表现力与高级气场。她可以轻扶手套、整理衣领、手插口袋或自然站立。不要舞台化，不要性感化，不要过度夸张。

## 8. 杂志版式、文字限制与最终效果

版式上加入醒目的虚构标题和高级时装专题栏目：可以使用 “ECLAT”“YUKINO YUKINOSHITA”“QUIET STRENGTH”“WINTER COUTURE”“MATERIAL FOCUS”“COORDINATE POINT”“THE NEW STANDARD” 等虚构文字。

注意这是虚构时尚杂志，不要使用真实杂志 logo，不要真实品牌标志，不要水印，不要真实杂志名，不要乱码，不要难以辨认的大段文字。整体效果要突出：雪之下雪乃、奢华高级、黑色长大衣、秋冬 couture、侧边服装材质栏目、平底鞋、无高跟鞋、真实杂志页面质感。

返工硬约束：不要出现任何随机数字、页码、期号、价格、年份、月份、日期、排名、序号、列表编号、条形码数字或无意义数字。侧边信息栏需要秩序感时，用短横线、色块、图标、无编号小标题和非数字项目符号替代。不要出现 01、02、03、Vol、No、2025、2026、980 yen 等数字文本。

镜头角度与姿势变化：人物采用低角度七分身或全身构图，身体微侧，脸略向下看镜头，一只手整理皮质手套或衣领，表现强气场。
