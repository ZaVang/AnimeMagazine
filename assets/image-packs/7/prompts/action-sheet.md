# Action Difference Sheet

Generate a clean action/pose difference sheet for the same character, outfit, and art style.

Use a simple grid layout with separated full-body or upper-body pose variants. Prefer neutral stand, slight turn, hand to lapel, holding bag if the outfit includes one, gentle walk step, and looking aside. Keep the same character, same outfit, same lighting, and clear whitespace between variants. Labels are optional; if labels appear, use short words and no numbers.

Common constraints:
- no real brand logos
- no watermark
- no unreadable large text blocks
- no random numbers, page numbers, issue numbers, prices, years, dates, rankings, serial numbers, barcode numbers, or meaningless numeric text
- keep the same character identity, outfit logic, color palette, and art direction from the source prompt


## Source prompt

## 1. 图像类型与角色锁定

请创作一张雨天都市时尚杂志内页风格的动漫插画，主角是《我的青春恋爱物语果然有问题》中的雪之下雪乃。

这张图要像真实日系时尚杂志中的雨季专题页面，而不是普通雨天角色插画。画面需要有都市 editorial、服装层次、版式留白和高级印刷质感。

## 2. 人物特征与气质表情

她有着黑色长直发、白皙肤色、精致清秀的五官、蓝灰色眼睛，整体气质要体现出安静、理性、清冷、优雅、克制和轻微疏离感。

表情可以是雨中回望、平静凝视或淡淡微笑，像在玻璃街区里从容停步。人物必须保持雪之下雪乃的辨识度，不要改变发色、眼色、脸型、发型长度或整体角色身份。

## 3. 杂志风格参考与主题方向

整体风格参考 SPUR、GINZA 和日系高级通勤时尚 editorial 的视觉气质，强调雨天质感、玻璃反射、清透冷调、实穿时装和都市女性的安静力量。

本页主题是“雨后的玻璃街区”。画面可以围绕东京现代商业街、玻璃幕墙、人行道积水、透明雨伞、橱窗倒影和柔和城市光展开。不要做成赛博朋克，不要夜店感。

## 4. 服装造型与鞋履禁忌

服装采用雨天都市通勤穿搭：防水质感长风衣、细针织高领、利落长裙或高腰直筒裤、轻便手提包、透明伞或深色长柄伞、简洁银色配饰。

整体搭配要实穿、成熟、干净，突出防水面料、风衣线条和针织柔软度。鞋子必须是平底防水短靴、平底切尔西靴、低矮方跟靴或简洁乐福鞋。禁止高跟鞋、细跟鞋、尖头高跟鞋、恨天高、夸张厚底鞋和过度性感鞋履。

## 5. 色彩、材质与季节感

色彩搭配建议以雨灰、玻璃蓝、墨黑、珍珠白、冷银色为主，局部可以加入淡紫、雾绿色或柔和蓝绿色反光。

材质需要清楚：防水风衣布、湿润石材、玻璃反射、伞面透明质感、针织纹理和哑光皮革。整体要湿润、清透、有雨后空气感，但不要脏乱。

## 6. 构图、背景与空间

画面采用竖版 9:16 或宽高比约 0.56 的杂志页面构图，适合 1200x2135 或更高分辨率。人物可以三分之二身或全身，站在玻璃墙前或撑伞回眸。

背景是雨后的东京玻璃街区，利用窗框、倒影和地面水痕形成自然分割。画面需要为标题、雨季穿搭说明和图注预留空间，不要让反射占满整页。

## 7. 光线、姿态与镜头质感

光线采用阴天自然光、橱窗反射和柔和环境光，面部光影细腻，衣服轮廓清楚，地面反光要有真实湿润感。

人物姿态要克制自然，可以撑伞、轻扶袖口、侧身回望或从玻璃门前走过。不要奔跑，不要夸张动作，不要过度可爱化。

## 8. 杂志版式、文字限制与最终效果

版式上加入杂志内页排版元素：雨季专题标题、穿搭小标签、图注块、细线分隔、少量英文与日文短句。可以使用虚构文字，例如“RAINY GLASS MODE”“AFTER THE RAIN”“YUKINO YUKINOSHITA”“雨の日の静けさ”。

注意这是虚构时尚杂志，不要使用真实杂志 logo，不要真实品牌标志，不要水印，不要乱码，不要难以辨认的大段文字。整体效果要突出：雪之下雪乃、雨天东京、玻璃反射、清冷通勤、平底靴、无高跟鞋、真实杂志内页质感。

返工硬约束：不要出现任何随机数字、页码、期号、价格、年份、月份、日期、排名、序号、列表编号、条形码数字或无意义数字。侧边信息栏需要秩序感时，用短横线、色块、图标、无编号小标题和非数字项目符号替代。不要出现 01、02、03、Vol、No、2025、2026、980 yen 等数字文本。

镜头角度、姿势与肢体自然度：人物正在玻璃门前缓步停下，透明伞微微向身后倾斜，身体朝画面深处迈步，前脚刚落在湿润地面上。一只手握伞柄，另一只手轻压风衣袖口或包带。脸可以在伞檐下呈三分之二侧脸，视线看向橱窗倒影或从镜头旁掠过。伞柄、手腕和肩膀形成顺畅线条，衣摆和发丝受雨后微风轻动，避免手臂穿过身体、伞面遮住脸或腿部步态别扭。
