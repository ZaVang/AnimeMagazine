# Background Only

Generate the same page, scene, or magazine layout without any person.

Reconstruct any area that would normally be hidden behind the character. Preserve layout, shadows, page texture, background architecture, material panels, color blocks, and editorial composition. Hard constraint: no human body parts, no silhouette, no ghosting, no hair, no hand, no shoes, no bag, no clothing fragments.

Common constraints:
- no real brand logos
- no watermark
- no unreadable large text blocks
- no random numbers, page numbers, issue numbers, prices, years, dates, rankings, serial numbers, barcode numbers, or meaningless numeric text
- keep the same character identity, outfit logic, color palette, and art direction from the source prompt


## Source prompt

## 1. 图像类型与角色锁定

请创作一张彩色学院派高级时尚杂志内页风格的动漫插画，主角是《我的青春恋爱物语果然有问题》中的雪之下雪乃。

这张图要像真实艺术时尚杂志中的学院派专题页面，而不是普通角色图。画面需要有摄影感、版式秩序、纸张颗粒和高级编辑质感，同时保持完整彩色画面。

## 2. 人物特征与气质表情

她有着黑色长直发、白皙肤色、精致清秀的五官、蓝灰色眼睛，整体气质要体现出知性、冷静、克制、清贵、安静而有距离感。

表情可以平静、专注、略带淡淡微笑，像在图书馆或展厅中接受一组文艺时装拍摄。人物必须保持雪之下雪乃的辨识度，不要改变发色、眼色、脸型、发型长度或整体角色身份。

## 3. 杂志风格参考与主题方向

整体风格参考艺术时装杂志和学院派时装摄影专题的视觉气质，强调学院派、书卷气、低饱和彩色、高级极简排版和安静的内在力量。

本页主题是“静默的学院派优雅”。画面可以围绕旧书墙、长桌、窗光、石膏像、黑板、白色窗框、极简图书馆或安静展厅展开，但不要变成校服写真。

## 4. 服装造型与鞋履禁忌

服装采用现代学院派时装造型：深色长款大衣、白衬衫、细针织开衫、丝巾或窄领带、高腰长裤或长款百褶裙、窄腰带、简洁皮包。

整体搭配要成熟、聪明、安静，不要学生制服化。鞋子必须是平底乐福鞋、平底牛津鞋、平底玛丽珍鞋或简洁平底短靴。禁止高跟鞋、细跟鞋、尖头高跟鞋、恨天高、夸张厚底鞋和过度性感鞋履。

## 5. 色彩、材质与季节感

画面以墨蓝、象牙白、冷灰、书页米色、深木色为主，可以加入少量冷蓝灰作为眼睛、标题线条或局部排版点缀。整体要低饱和、有彩色层次，不要做成黑白照片或纯灰画面。

材质需要清楚：羊毛大衣、棉质衬衫、细针织、哑光皮革、旧书纸张、木桌、石膏、深木色书架和窗光颗粒。画面需要有清楚明暗结构，但必须保留彩色杂志页质感。

## 6. 构图、背景与空间

画面采用竖版 9:16 或宽高比约 0.56 的杂志页面构图，适合 1200x2135 或更高分辨率。人物可以坐在长桌旁，也可以站在书架前，画面带大面积留白。

背景是极简图书馆、旧书墙或安静展厅。空间要有纵深，但不要堆满道具。左侧设置独立产品展示栏，不要做人物局部裁切解析。展示栏只陈列学院派单品本身，例如黑色乐福鞋、简洁皮包、丝巾或窄领带、折叠针织开衫、无文字旧书、皮带和布料小样，像杂志里的 product still life 页面，用细线和栏目留白增强杂志感。

## 7. 光线、姿态与镜头质感

光线采用有层次的彩色窗光，阴影清晰但柔和，突出脸部轮廓、发丝层次、衣物纹理、书页暖色和纸面颗粒。

人物姿态要端正、从容、安静，可以轻扶书页、倚在桌边、站在窗前、侧脸看向窗光或低头阅读，不必每次都看镜头。不要夸张摆拍，不要过度甜美，不要过度性感。

## 8. 杂志版式、文字限制与最终效果

版式上加入杂志内页排版元素：竖排标题、专题标签、英文栏目名、简短日文装饰文字、细线分隔和大面积留白。可以使用虚构文字，例如“QUIET STUDIES”“ACADEMIC MODE”“YUKINO YUKINOSHITA”“知性の余白”。左侧展示区以产品静物为主，背景干净，单品之间有留白和细线分隔；不要出现人物脸部、领口、手部、包带贴身处等局部截图，不要把左栏做成穿搭分解截图。

注意这是虚构时尚杂志，不要使用真实杂志 logo，不要真实品牌标志，不要水印，不要乱码，不要难以辨认的大段文字。整体效果要突出：雪之下雪乃、彩色学院派、知性冷感、墨蓝与书页米色、平底鞋、无高跟鞋、真实杂志内页质感。

返工硬约束：不要出现任何随机数字、页码、期号、价格、年份、月份、日期、排名、序号、列表编号、条形码数字或无意义数字。侧边信息栏需要秩序感时，用短横线、色块、图标、无编号小标题和非数字项目符号替代。不要出现 01、02、03、Vol、No、2025、2026、980 yen 等数字文本。

镜头角度、姿势与肢体自然度：人物坐在图书馆长桌一侧或半倚桌沿，不再正面站立。上半身微微前倾，像刚翻到一页停下来思考，手指轻压书页边缘，另一只手自然扶住桌沿或垂在膝侧。脸以三分之二侧脸朝向窗光，视线落在书页或略抬到画面外，不必看镜头。背部保持挺直但放松，肩膀不耸起，手肘有支点，膝盖和脚尖方向一致，避免悬空手臂、僵硬手指和不自然扭腰。

左侧产品展示硬约束：左栏只展示独立产品静物，不能使用人物身体局部截图。产品可以是黑色乐福鞋、皮包、丝巾或窄领带、折叠针织、皮带、无文字旧书、布料小样和色卡；每件产品要完整、清楚、像被摆在浅色纸面上拍摄，不要裁切人物皮肤或角色服装穿着状态。
