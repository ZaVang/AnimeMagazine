# Character Chroma Key

Generate only the character as a full-body standing sprite or clean pose subject.

The character must be on a perfectly flat solid #00ff00 chroma-key background. The background must be one uniform green color with no floor plane, cast shadow, contact shadow, gradient, texture, reflections, text, watermark, or decorations. Keep the full body visible from head to shoes with generous padding. Do not use #00ff00 anywhere in the character, outfit, eyes, accessories, or hair.

Common constraints:
- no real brand logos
- no watermark
- no unreadable large text blocks
- no random numbers, page numbers, issue numbers, prices, years, dates, rankings, serial numbers, barcode numbers, or meaningless numeric text
- keep the same character identity, outfit logic, color palette, and art direction from the source prompt


## Source prompt

## 1. 图像类型与角色锁定

请创作一张年轻文化与街头编辑感结合的日系时尚杂志内页，主角是《我的青春恋爱物语果然有问题》中的雪之下雪乃。

这张图要像真实年轻时尚杂志中的东京街角专题页面，而不是普通街拍海报。画面需要有城市文化感、干净版式、真实穿搭和纸质印刷质感。

## 2. 人物特征与气质表情

她有着黑色长直发、白皙肤色、精致清秀的五官、蓝灰色眼睛，整体气质要体现出聪明、冷静、安静、自信、清爽而不过分张扬。

表情可以自然放松，带一点淡淡微笑，像在东京街角进行一组轻量外套专题拍摄。人物必须保持雪之下雪乃的辨识度，不要改变发色、眼色、脸型、发型长度或整体角色身份。

## 3. 杂志风格参考与主题方向

整体风格参考 POPEYE、NYLON JAPAN 和日系年轻城市杂志的视觉气质，强调轻松街拍、文化场景、图形排版、年轻但不浮躁的时装感。

本页主题是“东京街角的轻量外套”。背景可以是白色街角、唱片店、画廊门口、地铁出口、低矮街树、自动贩卖机旁的干净墙面或晴天斑驳树影。

## 4. 服装造型与鞋履禁忌

服装采用轻松但精致的都市搭配：短款夹克、白衬衫、薄针织、宽松长裤、百褶长裙、帆布托特包、简洁斜挎包、金属小耳饰。

整体搭配要比前几页更年轻、更有街头文化气息，但仍然保持雪乃的克制和干净。鞋子必须是干净平底运动鞋、平底乐福鞋、低帮帆布鞋或平底短靴。禁止高跟鞋、细跟鞋、尖头高跟鞋、恨天高、夸张厚底鞋和过度性感鞋履。

## 5. 色彩、材质与季节感

色彩搭配建议以冷白、海军蓝、石板灰、灰绿、浅柠檬黄和少量红色小点缀为主。整体可以更轻快，但不要变成高饱和糖果色。

材质需要清楚：棉质夹克、轻薄针织、帆布、哑光皮革、街边水泥墙、玻璃门、树影和杂志纸面颗粒。颜色要有变化，避免只有蓝灰黑。

## 6. 构图、背景与空间

画面采用竖版 9:16 或宽高比约 0.56 的杂志页面构图，适合 1200x2135 或更高分辨率。人物可以全身出镜，站在街角或店门口，给标题和图注留出空间。

背景是明亮干净的东京街头，线条简洁，有店招、窗框或墙面图形作为版式辅助。不要过度拥挤，不要大量路人，不要让街头元素抢走人物。

## 7. 光线、姿态与镜头质感

光线采用明亮自然光，影子清楚但柔和，像晴天下午的城市街拍。人物和背景都要有真实空气感和印刷质地。

人物姿态要自然、轻松、从容，可以一手拿托特包、一手插口袋、靠近画廊门口行走、侧脸看向店窗或从镜头旁经过。不要夸张街舞动作，不要过度可爱化，不要校园制服感。

## 8. 杂志版式、文字限制与最终效果

版式上加入年轻杂志风格元素：竖排短标题、小标签、分栏说明、图注块和少量英文短句。可以使用虚构文字，例如“LIGHT OUTER GUIDE”“TOKYO CORNER”“YUKINO STYLE NOTE”“軽やかに、凛と”。

注意这是虚构时尚杂志，不要使用真实杂志 logo，不要真实品牌标志，不要水印，不要乱码，不要难以辨认的大段文字。整体效果要突出：雪之下雪乃、东京街角、年轻文化、轻量外套、平底鞋、无高跟鞋、真实杂志内页质感。

返工硬约束：不要出现任何随机数字、页码、期号、价格、年份、月份、日期、排名、序号、列表编号、条形码数字或无意义数字。侧边信息栏需要秩序感时，用短横线、色块、图标、无编号小标题和非数字项目符号替代。不要出现 01、02、03、Vol、No、2025、2026、980 yen 等数字文本。

镜头角度、姿势与肢体自然度：人物处在轻微行走中的街拍瞬间，身体斜穿过街角，不是静止站桩。一脚向前迈出，膝盖轻微弯曲，托特包自然贴在身侧晃动，另一只手可以插在夹克口袋或轻拉外套下摆。脸可以偏向店门玻璃或街树阴影，只留下半侧脸，不必回看镜头。肩、髋、腿的方向要符合真实行走，长发与外套有轻微动态，避免双脚平行僵站、手指无处安放或躯干过度扭转。
