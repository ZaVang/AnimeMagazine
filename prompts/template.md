# 8 段式时尚杂志 Prompt 模板

## 范式总结

前四个 prompt 的共同范式不是关键词堆叠，而是一份完整的时尚摄影 brief。它通常按这个顺序推进：先锁定图像类型和角色，再锁人物识别与气质，然后给出杂志参考、服装造型、色彩材质、构图背景、光线姿态、版式禁忌，最后用一组关键词收束整体效果。

这种写法的重点是让模型同时理解三件事：这是一张“真实杂志视觉”，主角必须是“雪之下雪乃”，画面气质要落在“日系高级时尚编辑”而不是普通动漫海报。

## 1. 图像类型与角色锁定

请创作一张[高质感日系时尚杂志封面 / 高端奢华时尚杂志封面 / 日系文化时尚杂志内页 / 真实杂志封底广告]风格的动漫插画，主角是《我的青春恋爱物语果然有问题》中的雪之下雪乃。

这张图要像真实出版物中的一页，而不是普通角色海报或手机壁纸。画面需要有完整的时尚编辑感、杂志设计感和高级印刷物质感。

## 2. 人物特征与气质表情

她有着黑色长直发、白皙肤色、精致清秀的五官、蓝灰色眼睛，整体气质要体现出知性、优雅、理性、克制、清冷而高级。

表情不要过于冷漠，可以是自然平静、略带淡淡微笑，展现安静、自信、聪明、从容的气质。人物必须保持雪之下雪乃的辨识度，不要改变发色、眼色、脸型、发型长度或整体角色身份，不要画成其他黑发少女。

## 3. 杂志风格参考与主题方向

整体风格参考[SO-EN / SPUR / GINZA / 国际高级时装大刊 / 日系文化时尚杂志 / 轻奢 editorial / 东京街拍专题 / quiet luxury back-cover advertisement]的视觉气质，强调[服装设计感 / 轻奢精致 / 东京都市感 / 文艺极简 / 奢华大片感 / 安静高级 / 知性女性气质]。

本页主题是“[主题名]”。主题可以围绕[东京书店 / 白色 atelier / 雨天玻璃街区 / 学院派图书馆 / 春季通勤 / 冬季 couture / 封底广告 / 服装结构专题]展开，但整体必须保持克制、干净、真实时尚杂志的审美。

## 4. 服装造型与鞋履禁忌

服装采用现代日系都市时尚穿搭，不走和风路线，不使用校服感造型。可以选择：剪裁利落的西装外套、短款结构夹克、长款风衣、修身高领针织衫、丝质衬衫、雾蓝细针织、高腰阔腿长裤、长款直筒裙、百褶长裙、窄腰带、简洁皮包、银色耳饰、珍珠或金属小配饰。

整体搭配要像真正的时尚编辑大片，突出服装材质、版型、轮廓和层次。鞋子必须是平底乐福鞋、平底牛津鞋、平底短靴、低矮方跟鞋、平底玛丽珍鞋或干净平底运动鞋。禁止高跟鞋、细跟鞋、尖头高跟鞋、恨天高、夸张厚底鞋和过度性感鞋履。

## 5. 色彩、材质与季节感

色彩搭配建议以[雾蓝 / 珍珠白 / 象牙白 / 浅灰 / 深海军蓝 / 炭黑 / 冷灰 / 香槟金 / 灰紫 / 浅米色 / 木色]为主，可以加入少量[银色 / 淡金色 / 酒红 / 灰绿 / 柔和蓝绿色]作为点缀。

整体色调要[柔和低饱和 / 高对比但克制 / 清透冷调 / 黑白高级 / 春季轻盈 / 冬季厚重 / 雨天湿润]，不要俗艳。材质需要清楚：羊毛、细腻针织、挺括西装面料、哑光皮革、轻薄衬衫布、玻璃反光、纸张颗粒和印刷质感。

## 6. 构图、背景与空间

画面采用竖版 9:16 或宽高比约 0.56 的杂志页面构图，适合 1200x2135 或更高分辨率。人物可以是[正面全身 / 七分身 / 三分之二身 / 坐姿 / 侧身回眸 / 背封小比例人物]，但必须是画面视觉主体。

背景可以是[高级摄影棚 / 东京街头 / 玻璃橱窗 / 安静书店 / 白色 atelier / 极简建筑外墙 / 雨天商业街 / 图书馆 / 封底广告留白空间]。整体构图要干净、平衡、有留白和编辑感，避免复杂喧闹。若是封面，可以让人物居中并形成强视觉中心；若是内页或封底，需要为标题、图注、条形码或说明文字预留空间。

## 7. 光线、姿态与镜头质感

光线采用[高级摄影棚 editorial 布光 / 柔和自然光 / 冬日冷调日光 / 雨天玻璃反射光 / 高反差黑白窗光]，强调人物面部轮廓、发丝、服装剪裁、布料层次和整体立体感。

人物姿态要自然、优雅、挺拔、从容，像真正的时装杂志模特。她可以[面向镜头 / 轻扶衣领 / 手插口袋 / 拿着书或杂志 / 站在玻璃窗前 / 侧身回眸]，但不要夸张摆拍，不要过度性感，不要动态过强。画面应有高级摄影与高质量动漫插画融合的质感。

## 8. 杂志版式、文字限制与最终效果

版式上加入杂志排版元素：顶部或侧边有醒目的虚构杂志标题，搭配简洁的小标题、栏目名、细线分隔、图注块、英文短句或日文短句。可以使用虚构文字，例如“ATELIER”“LUMIERE”“YUKINO YUKINOSHITA”“QUIET INTELLIGENCE”“SPRING MODE”“THE ART OF RESTRAINT”“雪ノ下 雪乃”。

注意这是虚构时尚杂志，不要使用真实杂志 logo，不要真实品牌标志，不要水印，不要乱码，不要难以辨认的大段文字。文字要少而有秩序，像真实杂志版式，而不是堆满画面。

整体效果要突出：[雪之下雪乃 / 日系高级感 / 时装编辑感 / 清冷知性 / 都市优雅 / 服装设计感 / 真实杂志封面感 / 纸质印刷感 / 无高跟鞋]。画风为高质量动漫插画，人物精致，服装质感明显，版型利落，画面高级且具有强烈视觉识别度。
